#  GeoFencePro 🌍🚧📍🛡️

#### How to Use :

1.</b> Register or Log in to the GeoFencePro web application.<br/>
2.</b> Set up a connection between the Sender (the tracked device) and the Receiver (the monitoring device) using a unique verification key.<br/>
3.</b> Define a Geofence around a specific area on the map by setting a perimeter in the "Create Fence" page.<br/>
4.</b> Start location tracking to share the real-time position of the Sender device.<br/>
5.</b> Receive real-time alerts automatically if the Sender breaches (enters or exits) the defined geofence boundary.<br/>

## Installation


### Pre-Requisites:
1. Install Git Version Control
[ https://git-scm.com/ ]

2. Install NodeJs
[ https://nodejs.org/en/download/ ]







### Clone the project:

```bash
  git clone https://github.com/saianand32/GeoFencePro.git

```

Go to the project directory

```bash
  cd GeoFencePro

```

**Backend Server:**

Go to server folder

```bash
  cd server
```
#### Backend:

Install all dependencies

```bash
 yarn install
```
Start the backend server

```bash
 yarn start
```


**Frontend Server:**

Open a new terminal and go to client folder

```bash
 cd client
```

Install all dependencies

```bash
 yarn install
```

Start the frontend server

```bash
 yarn start
```

#### Local Url for Server:

- Frontend : http://localhost:3000 
- Backend : Determined by the PORT in your `.env` file
